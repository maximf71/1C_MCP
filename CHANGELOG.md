# История изменений

## 0.6.0 — 2026-07-22

- добавлены `list_workspace_projects`, `ai_context` и агрегатор `code_search`;
- `write_module_source` получил dry-run по умолчанию, резервную копию, optimistic locking, семь режимов редактирования, ограничитель массового удаления и последующую диагностику EDT;
- добавлен `export_object` для EPF/ERF/CF/CFE с фиксированным каталогом результатов;
- добавлен ограниченный инструмент `git`: без shell-строк, произвольного репозитория и Git hooks, с обязательным подтверждением изменений;
- добавлены модульные и интеграционные тесты защитных ограничений.

## 0.5.0 — 2026-07-21

- добавлена интеграция Go-обёртки с DitriX EDT-MCP 2.8.1;
- `get_configuration_info` переведён на `get_configuration_properties` EDT;
- `get_object_structure` переведён на полный `get_metadata_details` EDT;
- добавлен ограниченный повтор вызова при временном построении проекта;
- добавлен опциональный нативный EDT bridge;
- локальные пути и имя проекта вынесены в параметры;
- добавлены инструкции установки, тестирования и устранения ошибок.
